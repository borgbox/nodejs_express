#NODE EXPRESS#

## Dependências ##

### Instalação express ###
* npm install --save-dev express

### Instalação nodemon ###
* npm install --save-dev nodemon

### Instalação EJS ###
* npm install ejs --save

### Instalação MYSQL ###
* npm install mysql --save

### Instalação Body-parser ###
* npm install body-parser --save

## Execução ##
### Executar em modo livereload ###
* nodemon app.js
